//-------------[Damage System]-------------//
[---{Hit Function}---]
Takes Character Ref Parameter
Takes Ref and Gets Health
Subtracts calls take Damage Function with Damage Value in Param

[---{Take Damage Function}---]
Takes Damage Param
Takes Health Value and subtracts it by Damage Param
Checks if Health is Greater Than 0
If so...
It will Update Health Percentage by Divided Min Health by Max Health
If not...
It will call Death Function

[---{Death Function}---]
Prints to Console -- [Character Name] is DEAD!

//-------------[Update Durabilty]-------------//
Checks if IsImmortal is True...
If IsImmortal is False...
Takes Damage Param
Takes Durability Value and subtracts it by Damage Param
Checks if Durability is Greater Than 0
If Durability is Greater Than 0...
Check if IsDynamic is True
If IsDynamic is True...
Multiply BaseDamageMulti by DurabilityPercentage and set EffectiveDamageMulti as the result
IsDynamic is False...
Do Nothing
If Durability is not Greater Than 0...
IsBroken = True
If Durability is not Greater Than or Less Than 0...
It will Update Durability Percentage by Divided Curr Durability by Max Durability
If IsImmortal is True...
Do Nothing

//-------------[Brake Handing]-------------//
When Character Hits, check if IsBroken is False
If False...
Apply Damage Multi
If True...
Don't Apply Damage Multi

||-------------[Basic Consumable Class]-------------||
//-------------[Consumable Values]-------------//
ConsumableName
Quantity
HealAmount
IsMedicine
CanOD
IsDeadly

//-------------[Heal Functionality]-------------//
Takes Character Ref
Takes Character Ref CurrHealth and adds it by HealAmount Value
Checks if Character Ref CurrHealth is Less Than Character Ref MaxHealth
If so...
It will add HealAmount Value to Character Ref CurrHealth then update the Character Ref HealthPercentage
If not...
Check IsMedicine...
If IsMedicine is True...
Check CanOD...
If CanOD is True...
Check IsDeadly...
If IsDeadly is True...
It will subtract the ExcessHealAmount from CurrHealth, death does apply, then update the Character Ref HealthPercentage
If IsDeadly is False...
It will set CurrHealth to Half of MaxHealth then update the Character Ref HealthPercentage
If CanOD is False...
It will set Curr Health to Max Health then update the Character Ref HealthPercentage 
If IsMedicine is False...
It will set Curr Health to Max Health then update the Character Ref HealthPercentage
